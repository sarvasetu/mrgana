OpenCode Zen is a curated AI gateway providing tested and verified models from OpenAI, Anthropic, Google, DeepSeek, Qwen, and more.

## Setup

1. Sign in to [OpenCode Zen](https://opencode.ai/auth) and get your API key
2. Configure Mrgana (API endpoint is auto-configured):

```bash
export MRGANA_LLM="opencode/gpt-5.4"
export LLM_API_KEY="your-opencode-zen-api-key"
```

That's it! The API endpoint is automatically set when you use an `opencode/` model prefix.

## Example Configurations

```bash
# OpenAI GPT 5.4
export MRGANA_LLM="opencode/gpt-5.4"
export LLM_API_KEY="your-opencode-zen-api-key"

# Claude Sonnet 5
export MRGANA_LLM="opencode/claude-sonnet-5"
export LLM_API_KEY="your-opencode-zen-api-key"

# Free model
export MRGANA_LLM="opencode/mimo-v2.5-free"
export LLM_API_KEY="your-opencode-zen-api-key"
```
